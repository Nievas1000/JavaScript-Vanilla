Tipos de memoria RAM:

RAM DDR:

La mayoría de las computadoras portátiles de antes de 2006 utilizaban una RAM DDR SO-DIMM de 200 pines. DDR significa Double Data Rate (Tasa de Datos Doble). Esto significa que la RAM transfiere datos en las transiciones del reloj 0-1 y 1-0, esencialmente produciendo el doble de la velocidad de transferencia de datos de las SDRAM (Synchronous Dynamic Random Access Memory). Este tipo de RAM estuvo disponible en tres diferentes configuraciones y velocidades: DDR226 PC2100, DDR333 PC2700 y DDR400 PC3200. La mayoría de las computadoras portátiles más viejas que tenían chips Celeron M, Centrinos antiguos o chips de AMD utilizaban RAM DDR333 de 200 pines.

RAM DDR2

La DDR2, que es la versión actualizada de la RAM DDR, mejoró al tener un rendimiento más rápido y más eficiente. Las DDR y DDR2 comparten características similares, pero la RAM DDR2 tenía más ancho de banda, permitiendo que más datos se transfirieran a través de la RAM al mismo tiempo. A pesar de que la DDR2 funciona más rápido que la DDR, esta utiliza menos memoria de la que usaba su hermana mayor. Otra diferencia entre las dos radica en el diseño físico de la DDR2, ya que esta no encaja en una ranura de RAM DDR. La DDR2 cuenta con un rango de velocidad de 400 MHz a 800 MHz. Las designaciones aparecen como DDR2-400 PC3200, DDR2-533 PC4200, etcétera.

RAM DDR3

La RAM DDR3, la hermana más rápida y joven de la DDR2, funciona a una velocidad entre 800 MHz y 1,6 GHz. Las nuevas computadoras portátiles soportan uno u otro tipo de RAM, pero no ambos. El tipo DDR3 no solamente incrementa la velocidad de la recuperación de información, sino que también reduce el consumo base de energía de la computadora en un 17%; tiene un prebúfer que carga 100% más rápido que el de la memoria DDR2 y usa la topología "Fly-by" para calibrar y controlar automáticamente el tiempo de latencia de la señal, eliminando las limitaciones físicas encontradas en la DDR2. La DDR3 no comparte las mismas conexiones de las patillas o las ubicaciones clave de la RAM DDR2, pero sí comparte el mismo perfil y apariencia de las DIMM.

Tipos de procesadores:

Microprocesador 8085

El microprocesador 8085 fue diseñado por Intel en el año 1977 con la ayuda de la tecnología NMOS.Las configuraciones de este microprocesador son el bus de datos de 8 bits, el bus de direcciones de 16 bits, que puede direccionar hasta 64 kb, contador de 16 bits y puntero de pila (SP). Los registros de seis bits están dispuestos en el par de BC, DE y HL. El microprocesador 8085 requiere una fuente de alimentación de 5 voltios.

CISC

Las órdenes se pueden ejecutar junto con otras actividades de bajo nivel. Realiza principalmente la tarea de cargar, descargar y recuperar datos en y desde la tarjeta de memoria. Aparte de eso, también hace cálculos matemáticos complejos dentro de un solo comando.Este procesador está diseñado para minimizar el número de instrucciones por programa e ignorar el número de ciclos por instrucciones. El compilador se utiliza para traducir un lenguaje de alto nivel al lenguaje de nivel ensamblador, porque la longitud del código es relativamente corta y se utiliza una memoria RAM adicional para almacenar las instrucciones.

RISC

RISC son las siglas de Reduced Instruction Set Computer y está diseñado para reducir el tiempo de ejecución mediante la simplificación del conjunto de instrucciones del ordenador.Estos tipos de chips se fabrican según la función en la que el microprocesador puede realizar pequeñas tareas dentro de un comando en particular. De esta manera, completa más comandos a un ritmo más rápido.En el microprocesador, cada conjunto de instrucciones requiere solo un ciclo de reloj para implementar el resultado en un tiempo de ejecución uniforme. Por lo tanto, reduce la eficiencia para más líneas de código, por lo que requiere una RAM adicional para almacenar las instrucciones. El compilador se utiliza para convertir el conjunto de instrucciones de lenguaje de alto nivel en un idioma informático.
